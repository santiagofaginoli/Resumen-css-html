CSS son las siglas en inglés para «hojas de estilo en cascada» (Cascading Style Sheets). Básicamente, es un lenguaje que maneja el diseño y presentación de las páginas web, es decir, cómo lucen cuando un usuario las visita. Funciona junto con el lenguaje HTML

Es tan importante ya que permite crear un Responsive Web Design, lo que significa que la página puede ser adaptada a cualquier dispositivo desde el cual se acceda.

Sintaxis de CSS

Selector: Es el elemento al que vamos a aplicar estilos.
Propiedad: Es eso que vamos a cambiar.
Valor: El nuevo valor que le vamos a dar a la propiedad.
Al conjunto de selector+declaración declaraciones se le denomina regla.

Box model es un módulo CSS que define cajas rectangulares. Cada vez que se inserta una etiqueta HTML, se crea una nueva caja rectangular que encierra los contenidos de ese elemento, este tiene distintas propiedades..

El margen es la separación entre una caja y las cajas adyacentes. El margen es la última área del Box Model y se puede ver como una separación invisible o transparente que ayuda a separar un elemento de otro.
Se pueden utilizar las siguientes propiedades:
margin-top = Margen superior.
margin-right = Margen derecho.
margin-top = Margen inferior.
margin-left = Margen izquierdo.

La propiedad padding establece el espacio de relleno requerido por todos los lados de un elemento. El área de padding es el espacio entre el contenido del elemento y su borde ( border ). No se permiten valores negativos. Se pueden utilizar las siguientes propiedades:
padding-top = padding superior.
padding-right = padding derecho.
padding-bottom = padding inferior.
padding-left = padding izquierdo.

La propiedad border permite definir en una única regla todos los bordes de los elementos seleccionados.
Se pueden utilizar las siguientes propiedades:
border-width = ancho del borde.
border-style = estilo del borde.
border-color = color del borde.
