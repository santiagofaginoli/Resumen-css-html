HTML no es un lenguaje de programación por si mismo, pero se lo considera un lenguaje de marcado de hipertexto( de ahi las siglas html), sirve para estructurar el contenido de una página web, es aquello que le da la forma y orden a la página, se podría decir que es su esqueleto.

Se puede utilizar listas ordenadas "ol", o listas desordenadas "ul", y para declarar los elementos que irán dentro de la listas se utiliza la etiqueta li.

HTML cuenta con diversos atributos, los más importantes son:
-Class: sirve para darle estilo a la página a través de CSS
-Title: este atributo se muestra como una descripción del elemento al que pertenece
-Data: permite guardar un valor en la etiqueta

En el formulario se agregan los diferentes campos de entrada de datos o de confirmación, así como los botones, que como mínimo ha de haber uno, el de envío

-form: etiqueta que engloba el formulario.
-label: sirve para escribir el nombre de campo para rellenar.
-input: sirve para crear un campo donde el usuario pueda interactuar.
-button: crea un botón para enviar el formulario.

La estructura del html

La estructura de html es como su esqueleto, en este se basara el resto de la pagina, este esta conformado por:
-Head: Se declara la información sobre el documento, como por ejemplo el titulo, enlaces a scripts y hojas de estilos.
-Body: Cuerpo de la página web.
-Footer: Pie de página, suele contener información sobre el autor y los creadores y la bibliografía o documentos relacionados.
-Article: Agrega información de otro sitio y que no pierda el sentido
-Section: Se utiliza para agrupar contenidos que tienen que ver con la pagina
-Aside: Contenido no originario de la página, como por ejemplo anuncios

Etiquetas mas importantes

Html se maneja mediante etiquetas, estas sirven para diversas funciones.Algunos ejemplos de estas etiquetas son:
H = Esta etiqueta esta para crear titulos; estos siguen un orden jerárquico, yendo desde el titulo mas importante "h1", hasta el titulo menos importante "h6".
p = Agrega parrafos que contienen texto, hasta no obtener los datos del cliente se suele utilizar un "Lorem ipsum" para rellenar.
a = Un ancla sirve para utilizar enlaces, los cuales son vinculos, que se pueden realizar tanto para vincular hojas de estilo como videos imagenes y hasta otras paginas.
